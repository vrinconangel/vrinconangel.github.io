Single Servo Motor Control

Materials:

  • Arduino Uno controller board(or similar)
  • 1 X Joystick Module
  • 1 X Servo Motor (SG90)
  • 7 X Jumper Wire
  • 4 X Female-to-Male wire

Setup: This project focuses on controlling a single servo motor with a joystick module, both of which are commonly found in Arduino starter kits. To begin gather all materials and set up the circuit according to the following diagram and connections explanation. The pin connections are as follows:

  • Joystick GND to A3
  • Joystick +5V to A2
  • Joystick VRx (Horizontal) to A1
  • Joystick VRy (vertical) to A0
  • Servo Orange wire to digital pin 9
  • Servo Red wire to 5V on Arduino
  • Servo Brown wire to GND on Arduino
